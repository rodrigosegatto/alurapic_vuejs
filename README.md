# alurapic

> Alurapic - projeto alura

## Build Setup VueJS

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Build Setup API Nodejs

``` bash
# access folder .\api\
cd api

# API with hot reload at localhost:3000
npm start

```
